# Skill Qualifier 🚀

Skill Qualifier is a full-stack web application designed to assess user skills based on role-specific questionnaires. It dynamically generates assessments based on selected skills and provides structured evaluation results.

---

## 📌 Features

- Role-based skill assessment generation
- Dynamic question loading from database
- Score evaluation system
- User authentication system (Google Oauth)
- REST API backend integration
- Responsive UI for all devices
- Smooth and modern frontend experience

---

## 🛠️ Tech Stack

**Frontend:**
- React.js / Vite
- Tailwind CSS / CSS Modules
- Axios

**Backend:**
- Node.js
- Express.js
- MongoDB (Atlas)
- Mongoose

**Deployment:**
- Frontend: Render
- Backend: Render
